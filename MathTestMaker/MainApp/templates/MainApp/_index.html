Math Test Maker

Fast. Free. Fancy.

What Is It?

Quickstart

You can make math exams here quickly. You just choose the types of questions you want on your test, select how many unique tests you want, specify points for different problems, and then choose a pretty format for the exams. Then you click a button and we generate exams and answer sheets you can download.

The idea is, your time is valuable and making math exams is menial labor. All the teachers across the world will be asking their students the same kind of questions. Find the x intercept of this line. Reduce this fraction. Compute the derivative of this function. Why should all the teachers be spending their time thinking about how to create a test for these things? There should be a go-to source for standard tests. This way you can get back to teaching and interacting with your students faster. Have more free time. Etc.

What Technology Is Used?

Probably put some pictures in here for the diff. technologies.

This project started out as a way to experiment with a bunch of cool differnt programming tools. The site runs on a digital ocean server and is powered by Bootstrap, Docker, Django, LaTeX, Python, Javascript, Stripe, !Mathematics! and more. It uses microservices and REST APIs. It's a pretty cool project from a let-me-use-as-many-cool-tools-as-possible perspective. And this leads us to our next point. . .

How Can I Contribute?

Are you a programmer? Are you a math teacher who wants to learn to program? Do you want to see some type of subject matter featured on this site? You can help! You can learn! This project is open to all interested parties. The project is hosted on Github . You can download the code that powers the site to play with it if you want. You can also write some code and contribute it!
